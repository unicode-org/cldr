Caveats

This is a beta version of the data, so there are known errors and omissions.

Updates

During the Beta period, the data will be updated as fixes are incorporated. Older data can be accessed through the CVS repository.

The purpose of the Common Locale Data Repository project is to:

  1. devise a general XML format for the exchange of culturally sensitive (locale) information for use in application and system development

  2. gather, store, and make available data generated in that format

On 2003.07.02, the specification for the XML format was released. As of 2003.10.31, the beta version of the locale data was made available for public review and feedback.

We request that people review the data for languages and countries that they are familiar with, and file bug reports where they see problems.

Comparison Charts. Perhaps the easiest way to provide feedback is to view the comparison charts, which provide an HTML view of data from different sources. There is also locale-insensitive supplementary data which is only available in XML format. That data tells which country uses which currency, how many digits each currency takes, and any special rounding behavior.

XML Source. The locale data XML specification and locale data XML files are checked into a CVS repository, currently hosted on the ICU site, and available via the web or by using CVS clients. The CVS tag is "release-1-0-alpha". For information on using CVS clients to access the repository, see readme.html.
