CLDR SurveyTool JavaScript Tests

  • in the 'js' directory, run npm i to install
  • To rebuild, run npm run build-test
  • If you want to continuously compile, keep npm run watch-test (control-C to cancel)
  • open the browser console to check for errors/warnings that are otherwise invisible!
