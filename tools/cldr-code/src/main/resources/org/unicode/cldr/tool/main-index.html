Caveats: These charts present some of the XML data in CLDR in a more readable form. However:

  • Not all of the data in CLDR is represented in the charts.
  • The data in the charts is based on the XML, but may have changes for readability. Do not assume that you can “scrape” the chart data and get correct CLDR data. Instead, access the source data directly, as described on CLDR Releases/Downloads.
  • For an interactive view of some of the CLDR data (with demos for collation and date patterns), please see ICU Locale Explorer. For example, see German. However:
    • ICU releases trail the CLDR releases, so for some period they show an earlier release.
    • ICU does not contain all of the data in CLDR.
    • The ICU Locale Explorer only shows a fraction of the locale data in ICU.
  • For information about the format and meaning of the charts, see Charts.
