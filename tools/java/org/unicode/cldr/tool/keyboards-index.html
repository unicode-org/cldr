Keyboard Charts

Main Charts Index

The keyboard charts provide a side-by-side comparison of data from different locales for each field.

  • characters → keyboards - These show a mapping of characters (and strings, in some cases) to keyboards that can generate them.
  • keyboards → characters - These show a mapping of keyboards to the characters that they generate, broken down by certain categories.
  • layouts - These show the different available layouts per locale.

For more details, see About Keyboard Charts.
